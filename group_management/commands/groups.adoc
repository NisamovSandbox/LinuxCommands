=== groups
El comando `groups` muestra todos los grupos a los que pertenece un usuario en Linux. Por defecto, muestra los grupos del usuario que ejecuta el comando, pero también se puede especificar otro usuario.

-Mostrar grupos del usuario actual::

[source,bash]
----
# Mostrar todos los grupos del usuario actual
groups
# Salida típica: users sudo developers
----

-Mostrar grupos de un usuario específico::

[source,bash]
----
# Mostrar todos los grupos del usuario "nisamov"
groups nisamov
# Salida típica: users sudo developers
----

-Diferencia con id::

[source,bash]
----
# 'groups' solo lista los nombres de los grupos
# 'id' muestra UID, GID y todos los grupos con sus IDs
id nisamov
----

-Usos prácticos::

[source,bash]
----
# Verificar permisos de acceso a recursos basados en grupos
# Comprobar si un usuario pertenece a un grupo con privilegios especiales
groups usuario
----

-Help:: Mostrar ayuda en pantalla

[source,bash]
----
# groups --help
# Muestra las opciones disponibles y la sintaxis de uso
----
