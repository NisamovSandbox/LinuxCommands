=== fdisk
Herramienta para crear, modificar y gestionar tablas de particiones en discos.

[source,bash]
----
# Abrir fdisk sobre un disco (modo interactivo)
fdisk /dev/sda
----

-l:: `fdisk -l` Listar tablas de particiones

[source,bash]
----
# Listar todos los discos y particiones
fdisk -l
----

-device:: `fdisk /dev/dispositivo` Seleccionar disco a gestionar

[source,bash]
----
# Gestionar particiones del disco /dev/sdb
fdisk /dev/sdb
