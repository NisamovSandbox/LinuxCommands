=== env
Mostrar el entorno de ejecución o ejecutar un comando con variables de entorno modificadas.

[source,bash]
----
# Mostrar todas las variables de entorno
env
----

VARIABLE=valor:: Definir una variable de entorno temporal

[source,bash]
----
# Definir una variable solo para este comando
VARIABLE=valor env
----

-i:: `env -i` Ejecutar un comando con un entorno vacío

[source,bash]
----
# Ejecutar un comando sin variables de entorno
env -i bash
----

-u:: `env -u` Eliminar una variable del entorno

[source,bash]
----
# Ejecutar un comando eliminando una variable concreta
env -u PATH bash
----

--:: Separar variables del comando a ejecutar

[source,bash]
----
# Ejecutar un comando con variables personalizadas
env LANG=C -- ls
----
