=== xdpyinfo
Mostrar información sobre la pantalla X y sus capacidades.

[source,bash]
----
# Mostrar información completa de la pantalla X
xdpyinfo
----

* -display:: `xdpyinfo -display DISPLAY` Especificar un servidor X concreto

[source,bash]
----
# Información de un servidor X específico
xdpyinfo -display :1
----

* -queryExtensions:: `xdpyinfo -queryExtensions` Listar las extensiones soportadas por X

[source,bash]
----
# Mostrar todas las extensiones de X
xdpyinfo -queryExtensions
----

* -queryFonts:: `xdpyinfo -queryFonts` Listar fuentes disponibles en el servidor X

[source,bash]
----
# Listar todas las fuentes disponibles
xdpyinfo -queryFonts '*'
----

* -version:: `xdpyinfo -version` Mostrar la versión del servidor X

[source,bash]
----
# Mostrar versión del servidor X
xdpyinfo -version
----
