=== xrandr
Configurar y consultar las pantallas conectadas y sus resoluciones en X.

[source,bash]
----
# Mostrar información de todas las salidas y resoluciones disponibles
xrandr
----

* -q:: `xrandr -q` Consultar el estado actual de las pantallas

[source,bash]
----
# Mostrar el estado actual
xrandr -q
----

* --output:: `xrandr --output OUTPUT` Seleccionar salida de pantalla

[source,bash]
----
# Activar o desactivar una salida específica
xrandr --output HDMI-1 --off
xrandr --output HDMI-1 --auto
----

* --mode:: `xrandr --mode RESOLUTION` Establecer resolución para una salida

[source,bash]
----
# Cambiar la resolución de una pantalla
xrandr --output HDMI-1 --mode 1920x1080
----

* --rate:: `xrandr --rate REFRESH_RATE` Establecer la frecuencia de refresco

[source,bash]
----
# Cambiar frecuencia de refresco de la pantalla
xrandr --output HDMI-1 --mode 1920x1080 --rate 60
----

* --rotate:: `xrandr --rotate ROTATION` Rotar la pantalla (normal, left, right, inverted)

[source,bash]
----
# Rotar la pantalla
xrandr --output HDMI-1 --rotate left
----

* --primary:: `xrandr --primary` Marcar la salida como principal

[source,bash]
----
# Establecer la pantalla principal
xrandr --output HDMI-1 --primary
----
