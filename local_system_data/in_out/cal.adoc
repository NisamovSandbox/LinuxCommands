=== cal
Mostrar calendarios en formato de consola.

[source,bash]
----
# Mostrar el calendario del mes actual
cal
----

-y:: Mostrar el calendario completo del año actual

[source,bash]
----
cal -y
----

-A:: Mostrar meses siguientes

[source,bash]
----
# Mostrar este mes y los 2 siguientes
cal -A 2
----

-B:: Mostrar meses anteriores

[source,bash]
----
# Mostrar este mes y los 2 anteriores
cal -B 2
----

-m:: Especificar el primer día de la semana (1=Lunes)

[source,bash]
----
# Comenzar la semana en lunes
cal -m
----
