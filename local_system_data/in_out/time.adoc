=== time
Medir el tiempo de ejecución de un comando, mostrando tiempo real, de usuario y de sistema.

[source,bash]
----
# Medir cuánto tarda un comando simple
time ls -l
----

-p:: `time -p` Formato POSIX más simple

[source,bash]
----
# Mostrar el tiempo en formato estandarizado POSIX
time -p ls -l
----

-o:: `time -o fichero.txt` Guardar la salida en un fichero

[source,bash]
----
# Guardar el resultado del tiempo de ejecución en un fichero
time -o resultado.txt ls -l
----

-a:: `time -a` Añadir al fichero en lugar de sobrescribir (junto con -o)

[source,bash]
----
# Añadir al final de resultado.txt en vez de sobrescribir
time -o resultado.txt -a ls -l
----
