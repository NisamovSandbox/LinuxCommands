=== dstat
Mostrar estadísticas combinadas del sistema en tiempo real (CPU, disco, red, memoria, procesos).

[source,bash]
----
# Mostrar estadísticas básicas en tiempo real
dstat
----

-c:: `dstat -c` Mostrar uso de CPU

[source,bash]
----
# Monitorizar uso de CPU
dstat -c
----

-d:: `dstat -d` Mostrar actividad de discos

[source,bash]
----
# Monitorizar lectura y escritura en disco
dstat -d
----

-n:: `dstat -n` Mostrar actividad de red

[source,bash]
----
# Monitorizar tráfico de red
dstat -n
----

-m:: `dstat -m` Mostrar uso de memoria

[source,bash]
----
# Monitorizar uso de memoria
dstat -m
----

-p:: `dstat -p` Mostrar estadísticas de procesos

[source,bash]
----
# Monitorizar procesos
dstat -p
----

-r:: `dstat -r` Mostrar estadísticas de E/S de disco detalladas

[source,bash]
----
# Monitorizar operaciones de disco
dstat -r
----

-y:: `dstat -y` Mostrar estadísticas del sistema (interrupciones y contexto)

[source,bash]
----
# Monitorizar interrupciones y cambios de contexto
dstat -y
----

--cpu --disk --net --mem:: `dstat --cpu --disk --net --mem` Mostrar varias métricas combinadas

[source,bash]
----
# Monitorizar CPU, disco, red y memoria simultáneamente
dstat --cpu --disk --net --mem
----

--top-cpu:: `dstat --top-cpu` Mostrar proceso con mayor uso de CPU

[source,bash]
----
# Mostrar proceso que más CPU consume
dstat --top-cpu
----

--top-io:: `dstat --top-io` Mostrar proceso con mayor uso de E/S

[source,bash]
----
# Mostrar proceso con mayor actividad de disco
dstat --top-io
----

--top-mem:: `dstat --top-mem` Mostrar proceso con mayor uso de memoria

[source,bash]
----
# Mostrar proceso que más memoria consume
dstat --top-mem
----

-t:: `dstat -t` Mostrar fecha y hora en la salida

[source,bash]
----
# Añadir timestamp a la salida
dstat -t
----

--output:: `dstat --output` Guardar salida en fichero CSV

[source,bash]
----
# Guardar estadísticas en un archivo CSV
dstat --output salida.csv
----
