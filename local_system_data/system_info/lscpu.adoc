=== lscpu
Mostrar información detallada sobre la arquitectura y la CPU del sistema.

[source,bash]
----
# Mostrar información básica de la CPU
lscpu
----

* -e:: `lscpu -e` Mostrar información por CPU/hilo (extendida)

[source,bash]
----
# Mostrar información detallada por núcleo e hilo
lscpu -e
----

* -a:: `lscpu -a` Mostrar CPUs offline y online

[source,bash]
----
# Mostrar todas las CPUs (incluidas las offline)
lscpu -a
----

* -b:: `lscpu -b` Mostrar tamaños en bytes (sin unidades humanas)

[source,bash]
----
# Mostrar tamaños en bytes
lscpu -b
----

* -c:: `lscpu -c` Mostrar el modo de compatibilidad

[source,bash]
----
# Mostrar compatibilidad de arquitectura
lscpu -c
----

* -J:: `lscpu -J` Salida en formato JSON

[source,bash]
----
# Exportar información de la CPU en JSON
lscpu -J
----

* -p:: `lscpu -p` Salida en formato parsable (CSV)

[source,bash]
----
# Mostrar información en formato CSV
lscpu -p
----

* -x:: `lscpu -x` Mostrar información extendida del layout

[source,bash]
----
# Mostrar topología detallada de la CPU
lscpu -x
----

* -y:: `lscpu -y` Mostrar información sobre hipervisor

[source,bash]
----
# Detectar si el sistema está virtualizado
lscpu -y
----

* --all:: `lscpu --all` Incluir CPUs offline (equivalente a -a)

[source,bash]
----
# Incluir CPUs offline
lscpu --all
----

* --online:: `lscpu --online` Mostrar solo CPUs online

[source,bash]
----
# Mostrar solo CPUs activas
lscpu --online
----

* --offline:: `lscpu --offline` Mostrar solo CPUs offline

[source,bash]
----
# Mostrar solo CPUs inactivas
lscpu --offline
----

* --help:: `lscpu --help` Mostrar ayuda del comando

[source,bash]
----
# Mostrar ayuda
lscpu --help
----

* --version:: `lscpu --version` Mostrar versión de util-linux

[source,bash]
----
# Mostrar versión de lscpu
lscpu --version
----
