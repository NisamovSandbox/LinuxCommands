=== lsmod
Mostrar los módulos del kernel cargados actualmente en el sistema.

[source,bash]
----
# Listar todos los módulos cargados
lsmod
----

* -n/a:: `lsmod` No tiene opciones propias, solo muestra Módulo, Tamaño y Uso

[source,bash]
----
# Salida típica
Module                  Size  Used by
nf_conntrack           16384  1
xt_conntrack           16384  1
iptable_filter         16384  1
----

* Uso combinado con otros comandos como `grep` para filtrar módulos

[source,bash]
----
# Buscar un módulo específico
lsmod | grep vfat
----

* Uso combinado con `modinfo` para obtener información detallada del módulo

[source,bash]
----
# Información de un módulo cargado
modinfo vfat
----
