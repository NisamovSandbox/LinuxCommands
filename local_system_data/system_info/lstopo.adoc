=== lstopo
Mostrar la topología del hardware del sistema (CPU, núcleos, memoria, caches, NUMA, etc.).

[source,bash]
----
# Mostrar la topología completa en consola
lstopo
----

* -p:: `lstopo -p` Mostrar IDs de procesador físicos (puertos físicos)

[source,bash]
----
# Mostrar IDs físicos de procesador
lstopo -p
----

* -c:: `lstopo -c` Mostrar topología en formato de consola compacta

[source,bash]
----
# Topología en consola compacta
lstopo -c
----

* -f:: `lstopo -f FORMAT` Exportar topología a diferentes formatos (png, pdf, svg, etc.)

[source,bash]
----
# Exportar topología a imagen PNG
lstopo -f png topologia.png
----

* --no-io:: `lstopo --no-io` No mostrar dispositivos de entrada/salida

[source,bash]
----
# Mostrar topología sin información de dispositivos IO
lstopo --no-io
----

* --whole-system:: `lstopo --whole-system` Mostrar todos los objetos del sistema

[source,bash]
----
# Topología completa del sistema
lstopo --whole-system
----
