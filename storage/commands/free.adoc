=== free
Mostrar el uso de memoria RAM y swap del sistema.

[source,bash]
----
# Mostrar uso de memoria en kilobytes
free
----

-h:: `free -h` Mostrar valores en formato legible

[source,bash]
----
# Mostrar uso de memoria en formato humano
free -h
----

-m:: `free -m` Mostrar valores en megabytes

[source,bash]
----
# Mostrar uso de memoria en MB
free -m
----

-g:: `free -g` Mostrar valores en gigabytes

[source,bash]
----
# Mostrar uso de memoria en GB
free -g
----

-s:: `free -s` Actualizar salida periódicamente

[source,bash]
----
# Actualizar cada 2 segundos
free -s 2
----

-c:: `free -c` Limitar número de actualizaciones

[source,bash]
----
# Mostrar 5 actualizaciones y salir
free -s 1 -c 5
----

--total:: `free --total` Mostrar total de memoria

[source,bash]
----
# Mostrar total de RAM y swap
free --total
----
