=== glxinfo
Mostrar información sobre la extensión OpenGL y el hardware gráfico.

[source,bash]
----
# Mostrar toda la información de OpenGL
glxinfo
----

* -B:: `glxinfo -B` Resumen de información básica de OpenGL

[source,bash]
----
# Información básica del renderizador OpenGL
glxinfo -B
----

* -i:: `glxinfo -i` Información sobre la tarjeta gráfica y la versión del driver

[source,bash]
----
# Mostrar información detallada del dispositivo y driver
glxinfo -i
----

* -l:: `glxinfo -l` Mostrar todas las extensiones soportadas

[source,bash]
----
# Listado completo de extensiones
glxinfo -l
----

* -c:: `glxinfo -c` Información del contexto GLX

[source,bash]
----
# Información del contexto
glxinfo -c
----
