=== xwininfo
Obtener información detallada de las ventanas en X11.

[source,bash]
----
# Ejecutar xwininfo e interactuar seleccionando una ventana con el cursor
xwininfo
----

* -root:: `xwininfo -root` Obtener información de la ventana raíz (pantalla completa)

[source,bash]
----
# Información de toda la pantalla
xwininfo -root
----

* -tree:: `xwininfo -tree` Mostrar jerarquía de ventanas

[source,bash]
----
# Mostrar árbol de ventanas
xwininfo -tree
----

* -stats:: `xwininfo -stats` Mostrar estadísticas detalladas de la ventana

[source,bash]
----
# Obtener estadísticas de una ventana seleccionada
xwininfo -stats
----

* -all:: `xwininfo -all` Mostrar toda la información disponible sobre la ventana

[source,bash]
----
# Información completa de la ventana
xwininfo -all
----

* -frame:: `xwininfo -frame` Incluir el marco de la ventana en la información

[source,bash]
----
# Obtener información incluyendo el marco
xwininfo -frame
----
