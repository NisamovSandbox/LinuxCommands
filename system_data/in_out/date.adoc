=== date
Mostrar o establecer la fecha y hora del sistema.

[source,bash]
----
# Mostrar la fecha y hora actuales
date
----

+FORMAT:: Usar formatos personalizados de salida

[source,bash]
----
# Solo año-mes-día
date +"%Y-%m-%d"

# Solo hora:minutos:segundos
date +"%H:%M:%S"
----

-s:: `--set` Establecer la fecha y hora (requiere permisos)

[source,bash]
----
# Establecer la fecha a 5 de febrero de 2026, 14:30
sudo date -s "2026-02-05 14:30"
----
