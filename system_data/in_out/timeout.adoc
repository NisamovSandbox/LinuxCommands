=== timeout
Ejecutar un comando durante un tiempo limitado; si se supera, se termina automáticamente.

[source,bash]
----
# Ejecutar un comando durante 5 segundos
timeout 5s ping -c 10 8.8.8.8
----

-k:: `timeout -k` Forzar el cierre del comando después de un tiempo adicional si no termina

[source,bash]
----
# Matar el comando después de 5s, forzando su terminación a los 7s si no finaliza
timeout -k 7s 5s ping -c 10 8.8.8.8
----

-s:: `timeout -s` Especificar la señal a enviar al finalizar el tiempo

[source,bash]
----
# Terminar el comando enviando SIGTERM (por defecto)
timeout -s TERM 5s ping -c 10 8.8.8.8
----

--foreground:: `timeout --foreground` Ejecutar en primer plano (útil con comandos que manejan terminal interactiva)

[source,bash]
----
# Mantener interacción con el comando en primer plano
timeout --foreground 10s top
----
