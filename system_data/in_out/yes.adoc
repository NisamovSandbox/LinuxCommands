=== yes
Generar repetidamente una cadena en la salida estándar, útil para automatizar confirmaciones o pruebas.

[source,bash]
----
# Repetir "y" continuamente (comportamiento por defecto)
yes
----

cadena:: `yes cadena` Repetir una cadena personalizada en lugar de "y"

[source,bash]
----
# Repetir "sí" continuamente
yes "sí"
----

Uso típico en scripts::

[source,bash]
----
# Automatizar la aceptación de prompts de un comando
yes | rm -i *.txt
----
