=== iostat
Mostrar estadísticas de CPU y rendimiento de dispositivos de almacenamiento.

[source,bash]
----
# Mostrar estadísticas básicas de CPU y discos
iostat
----

-c:: `iostat -c` Mostrar solo estadísticas de CPU

[source,bash]
----
# Monitorizar uso de CPU
iostat -c
----

-d:: `iostat -d` Mostrar solo estadísticas de dispositivos

[source,bash]
----
# Monitorizar actividad de discos
iostat -d
----

-x:: `iostat -x` Mostrar estadísticas extendidas de dispositivos

[source,bash]
----
# Monitorizar estadísticas avanzadas de discos
iostat -x
----

-k:: `iostat -k` Mostrar valores en kilobytes

[source,bash]
----
# Mostrar estadísticas en KB
iostat -k
----

-m:: `iostat -m` Mostrar valores en megabytes

[source,bash]
----
# Mostrar estadísticas en MB
iostat -m
----

-p:: `iostat -p` Mostrar estadísticas por partición

[source,bash]
----
# Mostrar estadísticas de todas las particiones
iostat -p ALL
----

-t:: `iostat -t` Mostrar fecha y hora en la salida

[source,bash]
----
# Añadir timestamp a la salida
iostat -t
----

-y:: `iostat -y` Omitir primer informe (desde arranque del sistema)

[source,bash]
----
# Mostrar solo estadísticas actuales
iostat -y
----

intervalo:: `iostat <intervalo>` Actualizar salida periódicamente

[source,bash]
----
# Actualizar estadísticas cada 2 segundos
iostat 2
----

conteo:: `iostat <intervalo> <conteo>` Limitar número de actualizaciones

[source,bash]
----
# Mostrar 5 actualizaciones cada 1 segundo
iostat 1 5
----
