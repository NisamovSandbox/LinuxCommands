=== uptime
Mostrar el tiempo que el sistema ha estado en funcionamiento, usuarios conectados y carga media.

[source,bash]
----
# Mostrar uptime simple
uptime
----

* -p:: `uptime -p` Mostrar tiempo de actividad de forma legible (pretty)

[source,bash]
----
# Mostrar tiempo de actividad legible
uptime -p
----

* -s:: `uptime -s` Mostrar fecha y hora de inicio del sistema

[source,bash]
----
# Mostrar fecha y hora de inicio
uptime -s
----

* -V:: `uptime -V` Mostrar versión del comando (opcional, dependiendo de la distro)

[source,bash]
----
# Mostrar versión de uptime
uptime -V
----
