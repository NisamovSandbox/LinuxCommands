=== dmesg
Mostrar y examinar los mensajes del *kernel ring buffer*.

[source,bash]
----
# Mostrar todos los mensajes del kernel
dmesg
----

* -T:: `dmesg -T` Mostrar marcas de tiempo legibles para humanos

[source,bash]
----
# Mostrar mensajes con fecha y hora legible
dmesg -T
----

* -H:: `dmesg -H` Salida paginada y más legible (human-friendly)

[source,bash]
----
# Ver mensajes de forma interactiva
dmesg -H
----

* -k:: `dmesg -k` Mostrar solo mensajes del kernel (equivalente a --kernel)

[source,bash]
----
# Filtrar solo mensajes del kernel
dmesg -k
----

* -l:: `dmesg -l NIVEL` Filtrar por nivel de severidad

Niveles habituales: `emerg`, `alert`, `crit`, `err`, `warn`, `notice`, `info`, `debug`

[source,bash]
----
# Mostrar solo errores y críticos
dmesg -l err,crit
----

* -f:: `dmesg -f FACILIDAD` Filtrar por subsistema/facilidad

[source,bash]
----
# Mostrar solo mensajes del subsistema usb
dmesg -f kern
----

* -w:: `dmesg -w` Mostrar mensajes en tiempo real (modo seguimiento)

[source,bash]
----
# Monitorizar mensajes del kernel en tiempo real
dmesg -w
----

* -C:: `dmesg -C` Limpiar el buffer de mensajes

[source,bash]
----
# Limpiar el buffer del kernel (requiere privilegios)
sudo dmesg -C
----

* -c:: `dmesg -c` Mostrar mensajes y limpiar el buffer después

[source,bash]
----
# Volcar mensajes y limpiar el buffer
sudo dmesg -c
----

* -x:: `dmesg -x` Mostrar nivel y facilidad en la salida

[source,bash]
----
# Mostrar mensajes con nivel y facilidad
dmesg -x
----

* --color:: `dmesg --color=auto` Activar salida coloreada

[source,bash]
----
# Mostrar mensajes con colores según severidad
dmesg --color=auto
----

* Uso típico:: Diagnóstico de hardware y arranque del sistema

[source,bash]
----
# Ver mensajes relacionados con discos
dmesg | grep -i disk
----
