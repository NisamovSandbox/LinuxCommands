=== ldd
Mostrar las dependencias de bibliotecas compartidas de un ejecutable.

[source,bash]
----
# Mostrar las bibliotecas dinámicas usadas por un binario
ldd /bin/ls
----

* -v:: `ldd -v` Mostrar información detallada (incluye versión de símbolos)

[source,bash]
----
# Mostrar dependencias con información detallada
ldd -v /bin/ls
----

* -u:: `ldd -u` Mostrar bibliotecas no utilizadas directamente

[source,bash]
----
# Mostrar dependencias no utilizadas
ldd -u /usr/bin/grep
----

* -d:: `ldd -d` Comprobar relocaciones de datos no resueltas

[source,bash]
----
# Verificar dependencias de datos
ldd -d /usr/bin/ssh
----

* -r:: `ldd -r` Comprobar relocaciones de datos y funciones

[source,bash]
----
# Comprobar todas las relocaciones
ldd -r /usr/bin/curl
----

* --version:: `ldd --version` Mostrar la versión de ldd

[source,bash]
----
# Mostrar la versión de ldd
ldd --version
----

* --help:: `ldd --help` Mostrar la ayuda del comando

[source,bash]
----
# Mostrar ayuda de ldd
ldd --help
----

* (sin opciones):: `ldd archivo` Analizar un ejecutable o biblioteca compartida

[source,bash]
----
# Analizar un ejecutable personalizado
ldd ./mi_programa
----

* Advertencia de seguridad:: No ejecutar `ldd` sobre binarios no confiables

[source,bash]
----
# Alternativa segura usando el cargador dinámico
LD_TRACE_LOADED_OBJECTS=1 ./binario_desconocido
----
