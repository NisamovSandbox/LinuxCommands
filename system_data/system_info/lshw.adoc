=== lshw
Mostrar información detallada del hardware del sistema, incluyendo CPU, memoria, buses, almacenamiento y dispositivos periféricos.

[source,bash]
----
# Mostrar un resumen del hardware
sudo lshw
----

* -short:: `lshw -short` Mostrar un resumen breve y legible

[source,bash]
----
# Resumen corto del hardware
sudo lshw -short
----

* -class:: `lshw -class CLASS` Mostrar solo una clase de hardware (ej. memory, cpu, storage)

[source,bash]
----
# Mostrar solo información de la memoria
sudo lshw -class memory
----

* -html:: `lshw -html` Exportar la información a un archivo HTML

[source,bash]
----
# Guardar información del hardware en HTML
sudo lshw -html > hardware.html
----

* -xml:: `lshw -xml` Exportar la información a XML

[source,bash]
----
# Guardar información del hardware en XML
sudo lshw -xml > hardware.xml
----

* -json:: `lshw -json` Exportar la información a JSON

[source,bash]
----
# Guardar información del hardware en JSON
sudo lshw -json > hardware.json
----

* -sanitize:: `lshw -sanitize` Eliminar información sensible (números de serie, MAC, etc.)

[source,bash]
----
# Mostrar hardware sin información sensible
sudo lshw -sanitize
----

* -quiet:: `lshw -quiet` Minimizar mensajes de advertencia

[source,bash]
----
# Ejecutar sin advertencias
sudo lshw -quiet
----

* -numeric:: `lshw -numeric` Mostrar IDs de clase y de bus en formato numérico

[source,bash]
----
# Mostrar IDs numéricos de hardware
sudo lshw -numeric
----

* --help:: `lshw --help` Mostrar ayuda del comando

[source,bash]
----
# Mostrar ayuda
lshw --help
----

* --version:: `lshw --version` Mostrar versión del comando

[source,bash]
----
# Mostrar versión de lshw
lshw --version
----
