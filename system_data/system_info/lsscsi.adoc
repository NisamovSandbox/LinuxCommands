=== lsscsi
Listar dispositivos SCSI y discos conectados al sistema.

[source,bash]
----
# Listar todos los dispositivos SCSI
lsscsi
----

* -g:: `lsscsi -g` Mostrar los archivos de dispositivo en /dev

[source,bash]
----
# Listar dispositivos con sus nodos en /dev
lsscsi -g
----

* -t:: `lsscsi -t` Mostrar tipo de dispositivo y controlador

[source,bash]
----

lsscsi -t
