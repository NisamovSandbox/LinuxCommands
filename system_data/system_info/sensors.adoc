=== sensors
Mostrar información de sensores de hardware (temperaturas, voltajes, ventiladores, etc.).

[source,bash]
----
# Mostrar todos los sensores detectados
sensors
----

* -u:: `sensors -u` Mostrar valores en formato "sin procesar" (raw)

[source,bash]
----
# Valores sin procesar de los sensores
sensors -u
----

* -f:: `sensors -f` Mostrar temperaturas en Fahrenheit

[source,bash]
----
# Sensores con temperaturas en Fahrenheit
sensors -f
----

* -A:: `sensors -A` Mostrar todas las lecturas incluyendo chips sin habilitar

[source,bash]
----
# Mostrar todas las lecturas posibles
sensors -A
----

* -s:: `sensors -s SENSOR` Establecer sensor específico para lectura

[source,bash]
----
# Leer un sensor específico
sensors -s coretemp-isa-0000
----
