=== vmstat
Mostrar estadísticas del sistema sobre memoria, procesos, CPU e I/O.

[source,bash]
----
# Mostrar estadísticas del sistema de manera resumida
vmstat
----

* -s:: `vmstat -s` Mostrar estadísticas de memoria en formato acumulativo

[source,bash]
----
# Estadísticas de memoria y sistema acumuladas
vmstat -s
----

* -m:: `vmstat -m` Mostrar información de slab cache

[source,bash]
----
# Información de slab cache
vmstat -m
----

* -d:: `vmstat -d` Mostrar estadísticas de discos

[source,bash]
----
# Estadísticas de discos
vmstat -d
----

* -p:: `vmstat -p [dispositivo]` Mostrar estadísticas de un dispositivo específico

[source,bash]
----
# Estadísticas de un disco concreto
vmstat -p sda
----

* [intervalo] [repeticiones]:: Actualizar cada cierto intervalo durante N repeticiones

[source,bash]
----
# Actualizar cada 2 segundos, 5 veces
vmstat 2 5
----
